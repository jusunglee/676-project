How can we connect people?

The more similar you are to someone, the easier it is to form a meaningful connection. [1]
The Myers-Briggs Personality Test is one way to group individuals on similarity.
Using machine learning, we can connect people to the same or similar MBTI personalities.


[1] “Is actual similarity necessary for attraction? A meta-analysis of actual and perceived similarity.” R. Matthew Montoya, Robert S. Horton, Jeffrey Kirchner

The Model

MBTI personality is composed of 4 different binary
sub-personalities.
‌
That means there are 8 sub-personalities and 16 distinct
personality combinations.
‌
A naive approach is to try a multi-modal 16-group classification.
However, this results in low performance because of large
number of clusters.
‌
Instead, let's create a model with 4 binary sub-classifiers
for each of the subpersonalities because Binary classifiers have higher
performance scores.
‌
In addition, this 4-bit model allows for "partial" personality matches, such as
Introverted vs Extroverted, instead of forcing a person to fit into one of
16 distinct categories and acting like clusters share no similarity.

Applications

Now that we have this ensemble model that can predict a MBTI, how do we use it?
‌
A user can classify themself with either the official MBTI test or our model.
‌
Then, reddit can utilize our algorithm to suggest users in the same classification.
‌
Since reddit already suggests people with similar interests (subredits), reddit can
now suggest people with similar personalities, closing the loneliness gap.
‌
While this model is trained on reddit data, you can trivially train the same model on
other social media platforms and utilize it there as well.
